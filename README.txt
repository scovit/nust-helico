This is nust-helico, a fork of NuST (http://www.lcqb.upmc.fr/nust/) for
Helicobacter. NuST has been describe in the following paper:

Vittore F Scolari, Mina Zarei, Matteo Osella and Marco Cosentino Lagomarsino.
NuST: analysis of the interplay between nucleoid organization and
gene expression. Bioinformatics. 2012 Jun 15;28(12):1643-4.
