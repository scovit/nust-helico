# TITLE
Genes with a paralog in Escherichia coli K12 from the Ensembl Bacteria database

# DESCRIPTION
List of all genes presenting at least a paralog in the dataset Escherichia coli K12 genes (EB 1 e_coli_k12) of Ensembl Bacteria ( http://bacteria.ensembl.org/index.html ). The list is limited to genes that have a RegulonDB gene ID.   

# REFERENCES
Kersey PJ et al. Ensembl Genomes: extending Ensembl across the taxonomic space. Nucleic Acids Res. 2010;38(Database issue):D563-9.
